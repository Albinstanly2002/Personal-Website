Fraud Guard
Credit Card Fraud Detection



Fraud Guard is a machine learning-based fraud detection system designed to identify fraudulent credit card transactions with high accuracy. Built using RandomForestClassifier, the model processes transaction data from uploaded CSV files and detects anomalies in real-time. It incorporates data preprocessing, feature scaling, and imbalanced data handling (SMOTEENN) to enhance prediction reliability.
